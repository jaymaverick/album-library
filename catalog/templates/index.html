Jay's Music Album Database

I got tired of keeping track of people borrowing my CDs, people returning my CDs, broken CDs, and overall which CDs I own.

So I developed this database as a Django project.

Welcome to Album Library, a grassroots Django website developed to catalog and keep track of my CD library.

It's also a proof of concept website, to show that I'm able to develop and deploy a Django-based website.

This website showcases basic use of views, models and templates, but also user authentication and connection to a MySQL database.

This website will remain as long as it's feasible as a portfolio example project.
